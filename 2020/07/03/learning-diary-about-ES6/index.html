今天是这两本书到的第二天，但是是第一天开始看，不过由于之前研究过var let const 所以关于这一部分很容易就能看明白，而且又相当于加深了一遍理解。下面记录一些之前没有彻底掌握的知识点或技术点

const命令

1. 首先基本用法就是用来声明一个常量，拿一个书上的例子来看吧

但是const本质并不是说定义之后便不可更改，而是其指向不可更改，对于常量，指向的内存地址保存的就是指向的内存地址，而对于对象和数组等数据，指向的就是指针，指针不会变，但指针指向的数据是可以变的。

2. 在浏览器端且在块级作用域内声明函数，就相当于var定义，因此会有类似var变量提升的效果，但限制只在于浏览器，不会在其他环境中存在。
3. do表达式
4. 暂时性死区。我对于暂时性死区的理解就是当处于块级作用域内并调用一个未声明的变量时，会触发错误的区域。也就是在这个块级作用域内，所有要使用变量在一开始就已经存在，但是处于‘未激活’状态，需要声明后将其‘激活’后才能正常使用。

关于上面的 3 和 4，目前还是没能彻底搞清楚，先记录一下。
