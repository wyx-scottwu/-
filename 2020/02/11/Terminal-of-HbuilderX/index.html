Terminal of HbuilderX 2020-02-11

Terminal not available of HbuilderX

If you find that the built-in terminal of HbuilderX is not available like me, and you cannot uninstall it unless you restart HbuilderX, then you’re in the right place, and I will show you how to handle the problem. Here we go!

  1. Open the HbuilderX installation directory
  2. Open main.js under HbuilderX/plugins/builtincef3terminal/script
  3. From line 17 to line 44, write the value in the shell as an absolute path, for example, change shell = “powershell.exe” to shell =’C:/WINDOWS/System32/WindowsPowerShell/v1.0/powershell.exe’
  4. shell=”cmd.exe” the same method
  5. Turn off the terminal and turn it back on.
