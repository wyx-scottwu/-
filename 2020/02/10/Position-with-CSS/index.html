Position with CSS 2020-02-10

I’m going to talk about position of CSS over here.

static（静态定位）

position属性的默认值，即无定位。使用static定位时，元素不会受到top，bottom，left，right影响。

relative（相对定位）

相对于无定位（静态定位）元素的定位。

absolute（绝对定位）

相对于最近的已定位父元素定位，如果没有已定位父元素，则相对于

fixed

相对于浏览器窗口定位，及时窗口滚动它也只会在固定位置。

sticky（粘性定位）

粘性定位的元素是依赖于用户的滚动，在 position:relative 与 position:fixed 定位之间切换。
它的行为就像 position:relative; 而当页面滚动超出目标区域时，它的表现就像 position:fixed;，它会固定在目标位置。
元素定位表现为在跨越特定阈值前为相对定位，之后为固定定位。
这个特定阈值指的是 top, right, bottom 或 left 之一，换言之，指定 top, right, bottom 或 left 四个阈值其中之一，才可使粘性定位生效。否则其行为与相对定位相同。
